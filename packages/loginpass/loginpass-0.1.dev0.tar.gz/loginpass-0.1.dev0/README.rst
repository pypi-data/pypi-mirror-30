Authlib Loginpass
=================

Social connections powered by Authlib_. This library is a part of Authlib project.
It works well with Authlib v0.7+.

.. _Authlib: https://authlib.org/


Features
--------

Connections that Authlib Loginpass contains:

- [x] Google
- [ ] GitHub
- [ ] Twitter
- [ ] Facebook
- [ ] Dropbox
