statr - trustworthy statistic in Python
=======================================

Building
--------

Call in `./source`::

  make header
  make library


Usage
-----
Example::

  import statr

  # try for example
  set_seed(10)
  statr.runif(10, [0, 9], 10)

Testing
-------

You can run example unit tests by::

  python test/uniform.py
