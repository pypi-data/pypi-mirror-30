Beschrijving Model
##################

Het Python package ``niche_vlaanderen`` laat de gebruiker toe om op basis van een aantal :doc:`invoer` de mogelijke vegetatietypes te berekenen.

De berekening gebeurt in drie stappen:

 * :doc:`trofie`
 * :doc:`zuur`
 * :doc:`vegetatie`
