## Description

Command line tool for project handling/building.


## Project file

### deploy: list

Can be:

* git
* pypi
* pypi-mw
* docker-mw
* helm

### dockerfiles: list

Path to each dockerfile. This can be used in case `deploy` is `docker`.

### helmChar: list

Path to each Helm chart. This can be used in case `deploy` is `helm`.
