Text Input
==========

The text input widget is a simple input field for user entry of text data.

.. figure:: /reference/images/TextInput.jpeg
    :align: center

Usage
-----

.. code-block:: Python

    import toga

    input = toga.TextInput(placeholder='enter name here')
