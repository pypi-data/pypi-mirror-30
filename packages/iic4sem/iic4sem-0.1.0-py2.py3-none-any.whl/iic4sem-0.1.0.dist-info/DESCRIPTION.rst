#### IIC4Sem

This project is for the academic requirement of semester 4 exams of IIC, UDSC.

#### Installation
To install this package, use pip.

```
pip install iic4sem
```

Then simply run `iic4sem` from the command line. 

#### What is missing?
* Unit Tests
* Project Work
* Question 5 - That has to be done in C
