For developer
=============

Requirements
------------

* pipenv
* make

Commands
--------

### Create and activate env

.. sourcecode::

    $ pipenv install -d
    $ pipenv shell

### Run

.. sourcecode::

    $ python jumeaux/executory.py <args>

### Serve docs

.. sourcecode::

    $ make serve-docs

### Unite test

.. sourcecode::

    $ make test

### Integration test

.. sourcecode::

    $ make test-cli


Version up
----------

There are 2 steps.

### Update release note (mkdocs/ja/releases/index.md)

.. sourcecode::

    $ make edit-release

### Confirm that your branch name equals release version

.. sourcecode::

    $ make release


Finally, create pull request and merge to master!!
